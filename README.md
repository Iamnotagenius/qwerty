# Qwerty
Check if a words from a set can be typed using only adjacent keys on a QWERTY keyboard.
To build it, simply:
```bash
make
```
Or, if you want to build it with `COMPILE_TIME_MAP` defined:
```bash
make COMPILE_TIME_MAP=1
```

## Usage
```bash
./a.out [-h] [-v | -q] [-i] [-s SEP | -0] FILE
```
